# 🍫 Chocolate Cake Recipe Page

Um projeto simples e elegante de **Página de Receita**, desenvolvido para praticar conceitos fundamentais de **HTML5** e **CSS3**, com foco em estruturação de conteúdo e organização visual.

Ideal para estudos de páginas informativas, blogs culinários e layouts básicos para websites.

---

## 📸 Preview do Projeto

Interface contendo:

- Título da receita
- Imagem ilustrativa do bolo
- Lista de ingredientes
- Estrutura limpa e organizada
- Layout simples e intuitivo
- Página centralizada para leitura fácil

---

## ✨ Tecnologias Utilizadas

- **HTML5**

---

## 🎨 Características do Design

- Visual minimalista

- Tipografia simples e legível
- Layout leve e funcional
- Interface amigável para leitura

## 🚀 Objetivo do Projeto

Praticar a criação de páginas estáticas utilizando HTML e CSS, explorando organização semântica de conteúdo e estilização básica para websites.

---

## 📚 Funcionalidades

- Exibição de imagem da receita
- Listagem de ingredientes
- Estrutura semântica em HTML
- Layout responsivo básico
- Fácil personalização de conteúdo
- Organização clara das informações
